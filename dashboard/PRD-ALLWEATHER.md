## 一、背景与目标

### 1.1 背景
CTA双均线策略在震荡市可能长期无开仓信号（如BTC震荡2个月），资金闲置。需要**All Weather全天候策略**作为补充，实现：
- **资金不闲置**: CTA空仓期配置All Weather
- **跨市场分散**: 美股/A股/港股/商品/数字货币
- **稳健收益**: 目标年化8-12%，最大回撤<20%

### 1.2 目标
开发完整的All Weather策略回测和监控功能，包含：
- 35+标的配置管理
- 再平衡提醒系统
- 与CTA策略的协同切换

**页面展示**:
- 目标配置饼图
- 实际持仓饼图
- 偏离度进度条（红色=超配、绿色=低配）

#### 2.2.2 再平衡提醒
**触发条件**:
- 单一标的比例偏离目标 ±5%
- 单一板块偏离 ±10%
- 时间触发：月度检查

**提醒内容**:
```
⚠️ 再平衡提醒
NVDA: 目标3.5% → 实际5.2% (+1.7%) [超配]
建议操作: 卖出 1.7% NVDA (~$1700)

贵州茅台: 目标1.5% → 实际0.8% (-0.7%) [低配]
建议操作: 买入 0.7% 茅台 (~$700)
```

#### 2.2.3 CTA协同切换
**状态显示**:
```
┌─────────────────────────────────────┐
│ CTA策略状态: 🟡 空仓观望 (已持续15天)  │
├─────────────────────────────────────┤
│ All Weather建议配置:                 │
│ ├── 当前配置: 20%                    │
│ └── 建议调整至: 80%                  │
│                                     │
│ [一键调整至建议配置]                  │
└─────────────────────────────────────┘
```

**切换规则**:
| CTA信号 | 持续天数 | All Weather配置 | CTA配置 |
|---------|---------|-----------------|---------|
| 金叉 | <7天 | 20% | 80% |
| 震荡 | >14天 | 80% | 20% |
| 死叉 | >7天 | 100% | 0% |

## 三、数据源需求

### 3.1 数据源清单

| 市场 | 数据源 | 状态 | 备注 |
|------|--------|------|------|
| 美股ETF | Yahoo Finance | ✅ 已有 | SPY/QQQ/GLD/NVDA等 |
| A股/港股 | FXI ETF代理 | ⚠️ 部分 | US-listed China ETF |
| 商品ETF | Yahoo Finance | ✅ 已有 | GLD/COPX/USO |
| 数字货币 | CoinGecko | ✅ 已有 | BTC/ETH/SOL |
| A股直接 | Tushare/AKShare | ❌ 待接入 | 可选增强 |

### 3.2 数据获取方案
**方案A: 代理ETF（当前实现）**
- A股: FXI (iShares China Large-Cap ETF)
- 港股: FXI + EWH (iShares MSCI Hong Kong)
- 优点: 数据易获取、美元计价
- 缺点: 不完全精确

**方案B: 直接API（未来增强）**
- A股: Tushare Pro / AKShare
- 港股: Yahoo Finance (.HK后缀)
- 优点: 精确跟踪
- 缺点: 需要额外API Key

## 五、API接口定义

### 5.1 新增API端点

```javascript
// 获取当前配置
GET /api/allweather/config
Response: {
  "targetAllocation": { "US": 0.35, "CN": 0.15, "HK": 0.10, ... },
  "currentAllocation": { "US": 0.38, "CN": 0.12, "HK": 0.11, ... },
  "deviation": { "US": +0.03, "CN": -0.03, ... },
  "positions": [...]
}

// 再平衡建议
GET /api/allweather/rebalance
Response: {
  "needsRebalance": true,
  "threshold": 0.05,
  "alerts": [
    { "asset": "NVDA", "target": 0.035, "current": 0.052, "action": "SELL", "amount": 1700 },
    { "asset": "贵州茅台", "target": 0.015, "current": 0.008, "action": "BUY", "amount": 700 }
  ]
}

// 跨市场相关性矩阵
GET /api/allweather/correlation
Response: {
  "markets": ["US", "CN", "HK", "COMM", "CRYPTO"],
  "matrix": [
    [1.0, 0.4, 0.6, -0.2, 0.3],
    [0.4, 1.0, 0.7, 0.0, 0.2],
    ...
  ]
}

// CTA协同信号
GET /api/allweather/cta-signal
Response: {
  "ctaStatus": "观望",
  "ctaDuration": 15, // 天数
  "currentAW": 0.20,
  "suggestedAW": 0.80,
  "suggestedCTA": 0.20,
  "reason": "CTA空仓已持续15天，建议增配All Weather至80%"
}
```
